# Privacy Policy

**Last updated**: 2026-04-11

Morning Korea Market Cast is a **personal automation tool** built and operated by Young-Won Kim for his own YouTube channel. This privacy policy describes how the tool handles data.

## Who this policy applies to

Only the developer (Young-Won Kim). **No other users authenticate with this application.** It is not a public service. It is not distributed. It runs on the developer's personal computer and uploads exclusively to the developer's own YouTube channel.

## What data the tool accesses

Morning Korea Market Cast accesses the following through the YouTube Data API:

- **The developer's own YouTube channel** — to upload videos the developer has created

The tool requests a single OAuth scope:

- `https://www.googleapis.com/auth/youtube.upload`

This scope grants permission to upload videos to the authorizing account's own YouTube channel.

## What data the tool does NOT access

- Other users' YouTube channels
- Viewer information (comments, subscribers, watch history)
- Private videos of other users
- Any personal data of third parties

## How data is used

- **OAuth access tokens and refresh tokens** are stored locally on the developer's machine and are used only to upload videos to the developer's own YouTube channel.
- **No data is shared with third parties.** No analytics, telemetry, advertising, or tracking is performed.
- **No data is sold.** The developer does not monetize any data.

## Storage

- OAuth tokens are stored locally in a file protected by standard file system permissions.
- Intermediate files (scripts, audio, images, finished videos) are stored locally on the developer's machine.
- No data is transmitted to any server owned by the developer other than Google's own YouTube Data API endpoints.

## Data retention

- OAuth refresh tokens are retained for as long as needed to operate the tool. They can be revoked at any time via [Google Account permissions](https://myaccount.google.com/permissions).
- Intermediate files older than 30 days are deleted locally.

## Compliance with Google API Services User Data Policy

Morning Korea Market Cast's use and transfer of information received from Google APIs adheres to the [Google API Services User Data Policy](https://developers.google.com/terms/api-services-user-data-policy), including the Limited Use requirements.

## Children's privacy

This tool is not intended for use by children under 13. It has no user base beyond the developer.
